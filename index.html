Typography Task Below

We were challenged to create 8 different photos for each type of typographical system using illustrator. Each system has its own seperate poster showing the differences between them, making these also helped create my Carp posters.

Axial: This involves one or more axis that the text follows and orientates itself around. Most of the time, it can be symmentrical, but as you can see here mine is not.

Bilateral: Bilateral designs are all over the place, nothing is symmetrical or centred, they end up looking like a big mess of text on a poster. On mine I wrote 3 different paragraphs and aligned them all differently.

Dilatational: Dilatational is designs using type set along circular paths, the text forms curves around a point. The text can also start wrapping a long one circle and then move a long to another.

Grid: Grid is pretty self explanatory. You design the layout of your page with a grid template with columns and/or rows. Even though you have to stick with the certain rules in the grid you can still be quite creative with it.

Modular: Modular uses repeated shapes to create a structure for your design. This gives you a lot more freedom compared to the likes of a grid layout. You are also not restricted to one axal.

Radial: Radial is similar to Dilatational, but the text can follow around more than one different circle.

Random: Random is in the name, place all the text in very random places, doesn't matter whether it is ledgible or not, the point of it being random is that it is all over the place.

Transitional: Transitional is text which follows very random lines, usaully towards the same direction or close to it, almost like waves.
